HOW TO GET INFORMATION FROM COMPUTER

Programmers keep track of their progress by looking at the console. The console is a test interface that programmers use for a variety of different purposes

Uses of console

The main use of the console is to output text from the program using code, more specifically a print statement.

Print statement

The print statement is a command that prints texts to the console for the programmers viewing pleasure, the print statement will vary depending on the language you are using.
Its for viewing and interpreting the computer's output from a program, the console is mainly a developer's tool and interacted with by the end user except in very abstract cases. e.g text based games, simple programs.

Characteristics

  • Tends to be hidden away behind the scenes
  • Dont try to implement the console in the final product
