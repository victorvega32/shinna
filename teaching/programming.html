MEANING OF PROGRAMMING

Programming can be defined as attempting to get a computer to complete a specific task without making mistakes.

Programming is a specific set of rules given to a computer in order to do a task right.

Machine code

Machine code is the only language the computer can understand, it is known as binary. A series of 1's and 0's.

Programming Languages

This is a tool used to translate our instructions into machine code[middle man].

  • Each programming language has its own unique uses
    • java and python: general level languages.
    • html/css: designed for specific uses
  • Each language also varies in how powerful it is
    • javascript isn't used for big problems.
    • java and python are
  • Each language also has an attribute known as power or level.
    • low level programming languages
      • assembly or c
    • high level programming languages
      • java or python

NOTE: The lower the level of your programming language the more your code will resemble what the machine can interprete as instruction{machine code}
